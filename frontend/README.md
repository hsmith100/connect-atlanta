# connect

Custom NextJS application built with Build-A-Server Burger

## Getting Started

This is a Next.js 14 application with proven chef special configuration.

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

### Build for Production

```bash
npm run build
npm start
```

### Stack

- **Frontend**: Next.js 14
- **Styling**: CSS Modules
- **Backend**: API
- **Database**: Database

## Health Check

Application includes health check endpoint at `/api/health` for container monitoring.
